# SnipSaver v0.5.4

- prevents clipboard self-saves by tracking the Windows clipboard sequence number
- keeps autostart conditional on the `autoStart` setting and installer choice
- writes both the Startup shortcut and the current-user Run entry when autostart is enabled
- removes visible `capture` wording from the UI text
- uses ASCII-stable language labels to avoid mojibake in Windows PowerShell 5.1
- adds a visible Windows 10 note for PNG clipboard retention
- loads fallback translations first so the selected/system language can override English
- reads JSON files explicitly as UTF-8 under Windows PowerShell 5.1
- switches language source to manual as soon as a concrete language is selected
- loads real `.ico` tray assets when available
